Buku Catatan Fisika

15 topik dasar fisika SMA, ditulis singkat dan padat. Gulir ke bawah untuk membaca, atau tanya di kotak bagian akhir — pilih mode Kata Kunci (instan, murni pencocokan kata) atau AI Lokal (model bahasa kecil yang berjalan langsung di browser Anda via WebLLM, gratis, tanpa API key, tanpa server).

Cari di Catatan

Pencari Kata Kunci

siap

Model bahasa kecil akan diunduh dan dijalankan langsung di browser Anda lewat WebLLM + WebGPU — bukan lewat server, tidak butuh API key, gratis sepenuhnya. Unduhan sekali saja lalu di-cache; setelah itu muat ulang jauh lebih cepat, bahkan bisa offline.

Bersiap mengunduh…
Contoh: "kecepatan dan percepatan" atau "gaya Coulomb". Mode Kata Kunci mencocokkan istilah fisika secara instan; mode AI Lokal bisa menjawab pertanyaan berkalimat lebih bebas setelah modelnya dimuat.

Mode Kata Kunci 100% lokal di browser — tidak ada AI, tidak ada API, tidak ada data terkirim ke server mana pun. Mode AI Lokal mengunduh model sekali dari CDN publik lalu semua percakapan berikutnya diproses di GPU perangkat Anda sendiri — tetap gratis, tanpa token, tanpa pembayaran.
